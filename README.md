# 🎂 Birthday Gift Website

A personalized birthday surprise website built using Django, HTML, CSS, and JavaScript.

## ✨ Features

- 🔐 Secret login page
- 🎁 Gift box reveal animation
- 🎵 Background music
- 💌 Secret letter with music
- 🎂 Cake cutting feature
- 🎮 Catch the Cake game
- 🎆 Fireworks and confetti effects
- 🌙 Dark / light theme switch
- 📱 Mobile responsive design
- 🚀 Hosted online using Render

## 🛠 Tech Stack

- Python
- Django
- HTML
- CSS
- JavaScript
- GitHub
- Render

## 📌 Project Purpose 

This project was created as a special birthday surprise website with animations, music, games, and personalized messages.

## ▶️ Run Locally

```bash
python manage.py runserver
```

Then open:

http://127.0.0.1:8000/
